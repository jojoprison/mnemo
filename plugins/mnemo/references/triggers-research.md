# Trigger Matrix — Research sessions

Read this file when review classifies the current session as **Research** (Read/Grep/WebSearch/Agent heavy, few writes).

| Signal | Recommended skill | Priority |
|--------|------------------|----------|
| Findings/discoveries not saved | mn:save | CRITICAL |
| No session summary created | mn:session | HIGH |
| New knowledge links to existing notes | mn:connect | MEDIUM |
| External articles / papers discovered | mn:save (as Source note) | HIGH |
| Multiple related findings surfaced | mn:save (as Molecule synthesizing them) | MEDIUM |
| PKM / vault patterns discovered (naming rules, note-type conventions, graph fixes) | mn:save (as Atom stating the rule) + update `references/` if it's a skill rule | HIGH |
